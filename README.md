# Captchonka

Captchonka is a pentesting tool to brute force captchas

Help:

`python captchonka -h` or `python captchonka --help`

Train:

`python captchonka --train "inputs/example.png"`

## Credits

Based on [CIntruder](https://github.com/epsylon/cintruder).

## Console colors

* *Red* - error
* *Green* - success
* *Blue* - info
* *Yellow* - warning
